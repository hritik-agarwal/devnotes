# Nginx

* Nginx is a web server used to server files.
* It is also used for other purposes such as 
  - Gateway
  - Reverse Proxy
  - Caching
  - Rate Limiting
  - Serving Static Sites
  - Host Multi Sites
  - Load Balancing
* In nginx.conf file, we have contexts and those contains directives.
  ```conf
  http {
    server 'nginx'
  }
  ```
